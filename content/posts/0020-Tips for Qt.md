# Tips for Qt

## Set a proper mirror for MaintenanceTool.exe

From: https://mirrors.tuna.tsinghua.edu.cn/help/qt/

```shell
.\MaintenanceTool.exe --mirror https://mirrors.tuna.tsinghua.edu.cn/qt
```
